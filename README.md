### Description

Simple CRUD rest API with Nest framework and mongodb. [Work in progress]

### Installation

```bash
$ npm install
```

### Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
